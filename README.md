# LLM AI Service - Corvus Platform

Microservicio encargado de la integración con motores de Inteligencia Artificial locales (Ollama - Llama 3.2 3B) y en la nube (GroqCloud - Llama 3.3 70B). Procesa prompts para evaluación académica, análisis de océanos azules y heurísticas de innovación.
